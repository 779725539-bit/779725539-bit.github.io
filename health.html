当前推荐链路

先让 Health Auto Export 导出 JSON 到 iCloud Drive，Mac 同步后由 OpenClaw 读取。跑稳之后，再升级 REST API 或 MCP/TCP。

  1. iPhone 导出到 iCloud Drive / Health Auto Export / openclaw-health
  2. Mac 同步到本地文件夹
  3. OpenClaw 生成本机健康台账与公开摘要
  4. 本地脚本只发布脱敏摘要到 GitHub Pages

REST API 怎么放

GitHub Pages 不能接收 POST。REST API 要放到 Cloudflare Worker、Vercel Function、Supabase Edge Function 或自己的服务器上，然后网页读取它生成的公开摘要。

如果要手机离开局域网也能同步，下一步用 HTTPS 接收接口；如果只在家里同步，局域网 Mac 接收器也可以。

隐私边界

公开页面只适合放趋势和结论，不适合放原始健康数据。运动路线、用药、症状、ECG、情绪状态、逐点心率默认不发布。
