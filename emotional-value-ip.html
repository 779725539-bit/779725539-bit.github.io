01 总纲正文入口

功能让人开始用，
情绪让人留下来

一个强 C 端产品，往往不是功能更强，而是能把功能、情绪、身份和关系揉在一起。

01

功能

解决外部任务：它帮我完成什么。

02

情绪

解决内部体验：它让我处在什么状态。

03

IP

不是故事本身，而是用户借来表达自己的符号。

04

陪伴硬件

低压力、稳定在场，替用户记住生活。

功能决定下限，情绪决定溢价；工具解决问题，关系创造留存。
02 情绪价值概念对照

情绪价值
不是功能的反面

好功能会长出情绪价值，情绪价值也可以成为一种功能。

功能价值

偏外部任务。

它回答的是：这个东西帮我完成什么？

例如 Codex 把 bug 修好，这是功能。

情绪价值

偏内部体验。

它回答的是：它让我感觉自己处在什么状态？

连续几次靠谱之后，“有人兜底”的安心感就出现了。

安心被陪伴被理解掌控感身份感有趣有希望仪式感
摆件、潮玩、香水、手账看似“没用”，但能稳定情绪、表达身份、保存记忆。
03 C 端产品内容化表达

To C 的本质
是做内容

C 端用户没有义务理解你。你要先抢到注意力，再制造感受、记忆、欲望和分享冲动。

无聊用户不会替产品找意义。
不酷没有身份感，就难以被展示。
不值价格之外，还要有心理账。
不方便低摩擦是 C 端底线。
产品本身要像内容一样可感知，卖点要能被一句话传播。
使用过程要产生可晒、可讲、可复述的瞬间；品牌要持续输出故事、审美、身份和情绪。
04 购买动机8 格卡片

用户买的
不是物品本身

用户买的是情绪、审美、陪伴、收藏感、社交货币、低门槛艺术、礼物价值和身份表达。

买情绪这个东西有点像我
买审美这是我的品味
买陪伴它没什么用，但它在那儿
买收藏感我想集齐，想拥有稀缺款
买社交货币懂的人自然懂
买低门槛艺术我也能拥有一点艺术感
买礼物价值这个礼物轻，但不敷衍
买身份表达它替我说明我是什么样的人
低价情绪符号 + 可展示收藏 + 场景纪念 + 持续上新，是很多消费品的底层组合。
05 新 IP新旧路径对比

过去 IP 是故事角色，
现在 IP 是情绪形状

新 IP 不一定先让用户看完故事，而是先用一个强视觉符号占住用户的眼睛和情绪。

过去路径

故事理解角色建立感情买周边

现在路径

形象一眼记住情绪投射展示收藏反过来补故事
Labubu 先作为强视觉符号进入生活，让用户觉得：这东西有点像我。
06 方法论7 个变化 + 5 问

判断新 IP，
先问5 个问题

新 IP 的关键不是完整世界观，而是能不能被一眼记住、被展示、被二创、被持续上新。

新 IP 的 7 个变化
01从故事 IP 到情绪 IP
02从角色设定到视觉记忆点
03从内容先行到商品 / 社交先行
04从大众共同观看到算法反复触达
05从英雄叙事到低能量共情
06从公共神话到个人配饰
07从民族文化输出到轻文化负担输出

01它占据什么新情绪？
02它能不能一秒被记住？
03它能不能被展示？

新 IP 更像小型精神图腾、社交暗号、情绪配饰和可变体符号系统。
07 符号传播传播动机

人不是在传播 IP，
而是在传播自己

符号会被传播，不是因为符号自己厉害，而是因为传播它对用户有用。

表达自己

表达身份：我是谁，我是什么审美和圈层
提升形象：给自己贴标签
替我表达情绪：委屈、反卷、怪可爱
投射理想自我：靠近想成为的人

进入关系

寻找同类：懂的人自然懂
制造谈资：热门 IP、明星、梗
获得参与感：加入集体事件
建立关系：送周边、转梗、一起讨论

获得外部收益

证明在场：我知道这个热点
降低表达成本：一句歌词、一个表情包
合理化消费：晒出来，说限量
获得流量：内容素材带来互动
一个符号越能帮用户表达自己，越容易被用户主动传播。
08 情绪坑位三大地图

新 IP 抢的
不是故事，是情绪坑位

当代用户的情绪需求，可以先归成三大类：安全感缺失、表达欲爆发、关系重新组织。

安全感缺失

累、焦虑、不确定，想抓住一点稳定感。

  • 疲惫感
  • 委屈感
  • 确定感 / 好运
  • 自我管理焦虑

表达欲爆发

不想再按传统体面活，想表达真实状态。

  • 反卷感
  • 发疯感
  • 低成本体面
  • 复古怀旧
  • 本土文化自信

关系重新组织

不想高压力社交，但仍需要陪伴和同类。

  • 陪伴感
  • 身份暗号
  • 私人化
现在的新 IP，不是在讲“快乐人生”，而是在替用户表达：我很累，但我还想可爱。
09 xiwa / 挂件产品落地

陪你经历，
也替你记得

xiwa 不应该被表达成一个硬聊天 AI，而是一个低压力、稳定在场、能陪用户经历时间的小存在。

不是硬聊天陪伴不等于一直对话。
低摩擦在场随身、轻触、随取随用。
状态符号把生活碎片变成小怪兽。

xiwa｜一个会听你说话的潮玩手机挂件

现代人的生活太碎了：情绪碎、灵感碎、关系碎、任务碎、身体状态碎。

手机能记录，但它太主动。要掏出来、打开、组织语言。

所以用户会被 always on、低摩擦、随身在场的设备吸引。

困困怪：昨晚睡得少，今天慢一点
纸条怪：事情有点多，先留三张
电池怪：恢复值低，别硬扛
星星怪：有一个小成就
躲躲怪：今天想逃避一下
主角不是功能人格，而是情绪容器。小怪兽不是任务模块，而是状态符号。

12 Checklist执行检查

判断一个情绪产品
能不能成立

以后判断 IP、挂件、陪伴产品、情绪消费品，都可以先看这页。

先看情绪
再看符号
再看传播
最后看陪伴

新 IP 5 问

  1. 占据什么新情绪？
  2. 能不能一秒被记住？
  3. 能不能被展示？
  4. 有没有二创空间？
  5. 能不能持续上新？

符号产品 6 问

  1. 我自己有感觉吗？
  2. 别人能读出我是谁吗？
  3. 能佩戴、展示或分享吗？
  4. 有没有稀缺机制？
  5. 有传播机会吗？
  6. 能成为社交暗号吗？

陪伴产品 5 问

  1. 是不是硬聊天？
  2. 是否低压力？
  3. 是否稳定在场？
  4. 是否能和时间绑定？
  5. 是否像我的小角色？

挂件方向

  • 情绪见证：生日、失恋、上岸、发财、出差、城市限定
  • 轻陪伴：摸一下会亮、每天一句话、说辛苦了
  • 状态符号：困困怪、纸条怪、电池怪、星星怪
