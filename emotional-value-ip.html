02 总命题价值迁移

功能让人开始用，
情绪让人留下来

人的消费和使用从来不是纯理性的。强产品通常同时拥有功能、情绪、符号和关系。

01功能价值解决问题，决定下限。
02情绪价值制造感受，决定溢价。
03符号价值表达身份，制造关系。
04关系价值长期陪伴，形成留存。
功能解决问题，情绪创造关系。
03 功能与情绪不是对立

好功能会长出情绪，
情绪也能成为功能

边界不是“有用 / 没用”，而是外部任务和内部体验的不同。

功能价值

偏外部任务：它帮我完成什么。

例子：修好 bug、记录内容、跑通流程、稳定使用。

连续几次靠谱
就会变成
“有人兜底”
情绪价值

偏内部体验：它让我感觉处在什么状态。

安心、被陪伴、被理解、有掌控感、有身份感、有希望。

白家电式的稳定感，也是功能长出的情绪价值。
04 情绪价值执行什么无用之用

“没用”的东西，
执行的是心理功能

摆件、潮玩、香水、手账、AI 陪伴，看似不是工具，但它们并非没有功能。

稳定情绪让状态被安放。
表达身份替我说明我是谁。
保存记忆把经历留在物上。
建立仪式感让日常变得可感知。
安心有人兜底。
陪伴它没什么用，但它在那儿。
掌控感生活变得有秩序。
希望感我还可以变好。
情绪价值不是功能的反面，它是心理性的、社会性的功能。
05 To C 本质注意力战场

To C 的本质，
是做内容

C 端用户没有义务理解你。你要先抢到注意力，再制造感受、记忆、欲望和分享冲动。

用户怕什么无聊、不酷、不值、不方便。
主战场不是功能说明，而是注意力。
可感知产品本身像内容。
可传播卖点能一句话说清。
可复述使用过程可晒、可讲。
可持续品牌持续输出故事、审美、身份和情绪。
C 端很多产品表面是工具，底层都在做内容化表达。

功能很弱，但情绪很强；它不解决问题，却制造意义、身份、陪伴、收藏欲和表达欲。

07 用户购买动机8 种购买

用户买的不是物品本身

同一个潮玩、IP 商品或情绪符号，用户可能同时买了八种东西。

买情绪这个东西有点像我。
买审美这是我的品味。
买陪伴它没什么用，但它在那儿。
买收藏感我想集齐，想拥有稀缺款。
买社交货币懂的人自然懂。
买低门槛艺术我也能拥有一点艺术感。
买礼物价值这个礼物轻，但不敷衍。
买身份表达它替我说明我是什么样的人。
低价情绪符号 + 可展示收藏 + 场景纪念 + 持续上新，是很多消费品的底层组合。

10 IP 形态变化叙事到投射

过去 IP 是故事角色，
现在 IP 是情绪形状

新 IP 不一定先让用户看完故事，而是先用强视觉符号进入生活。

过去路径

01先有故事
02用户花时间看
03理解角色
04建立感情
05买周边

现在路径

01先有形象

过去 IP 是大众故事的角色，现在 IP 是个人情绪的形状。

13 新 IP 7 变时代变化

新 IP 的变化，
不是只换画风

它从长叙事、公共神话，变成低解释成本、强视觉、强投射的情绪符号。

故事 IP → 情绪 IP先占住一种情绪。
设定 → 视觉记忆点用户没耐心先学世界观。
内容先行 → 商品 / 社交先行先进入生活。
共同观看 → 算法触达平台和粉丝先选中。
英雄叙事 → 低能量共情被理解、被安抚、被允许不成功。
公共神话 → 个人配饰身上的小信号。
宏大文化 → 轻文化负担低解释成本情绪符号。
代表Labubu、Chiikawa、卡皮巴拉。
全球化不一定靠宏大文化，也可以靠低解释成本的情绪符号。

判断新 IP，

它占据什么新情绪？
  • 反卷、焦虑、发疯

它能否一秒被记住？
  • 轮廓

它能否被展示和二创？
  • 包上、桌上、头像

16 陪伴层级不是一直对话

陪伴不等于
一直对话

陪伴可以从静态存在，到情绪见证，再到轻互动，最后到人格陪伴。

静态陪伴它不需要回应，只要在那儿。
情绪见证陪用户经历时间和人生节点。
轻互动陪伴给一点点回应，但不是高强度聊天。
人格陪伴像“我的小角色”。
很多陪伴产品真正卖的是低压力关系：不打扰、不评判、不索取注意力。

18 三大集体情绪12 个坑位

12 个情绪坑位，
来自三大集体情绪

上层是集体情绪，下层才是具体坑位。

安全感缺失

累、焦虑、不确定，想抓住一点稳定感。

疲惫感委屈感确定感 / 好运自我管理焦虑
表达欲爆发

不想再按传统体面活，想表达真实状态。

反卷感发疯感低成本体面复古怀旧本土文化自信
关系重新组织

不想高压力社交，但仍需要陪伴和同类。

陪伴感身份暗号私人化
现在的新 IP，不是在讲快乐人生，而是在替用户表达：我很累，但我还想可爱。
