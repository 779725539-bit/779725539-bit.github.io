01 / 08 因为独特

泡泡玛特的独特方法论

从濒临倒闭的杂货铺到千亿市值的潮玩帝国——王宁的商业智慧不在于追逐风口，而在于坚守独特。

🎨
IP 即核心资产
用唱片公司理解泡泡玛特
IP 的本质是低成本的市场营销。泡泡玛特不是玩具公司，是 IP 平台——像唱片公司签约艺术家一样签约设计师。
⛪
门店即教堂
线下传递品牌文化
线上各品牌很难有差异，但线下门店能调动视觉、触觉、听觉与服务，全面传递品牌文化。门店是品牌的教堂。
✨
满足感 + 存在感
消费的两个本质
消费就是解决两个问题：满足感和存在感。泡泡玛特解决的是情感慰藉需求——不是生活刚需，是精神刚需。
02 / 08 因为独特

王宁的经营哲学

"
本来想做 A，然后做了 B，最后在 C 上伟大。
ABC 背后都有一个共同目标——满足人的快乐、分享欲和收藏欲。
—— 王宁，泡泡玛特创始人
  • 01

    尊重时间，尊重经营

    不追风口，不做打折促销。在所有人冲向移动互联网时，王宁还在琢磨收银员应该坐着还是站着、音乐应该开多少分贝。

  • 02

    每个小品类都可以诞生伟大的公司

    不追求做大众消费品，而是做收藏品。在看似天花板很低的品类里，用深度和独特性撑起千亿市值。

  • 03

    全球化 × IP 集团化

    双引擎战略：国际业务扩展（尤其非华人市场）+ 围绕 IP 的多业务线延展。"道"没变，变的是"术"。

04 / 08 真需求

什么是真需求？

"
你能连接的人，不是你的人脉；
你能帮到的人，才是你的人脉。
—— 梁宁
真需求 = 刚性
有些东西你一直在吐槽，一直在怨念，却无法离开。最核心的原因是需求——你对它的需求依然存在。真需求，就是这么刚性。
❌
伪需求的特征
看着挺好，但是没有也可以。加了微信、吃过饭、点过赞——如果对方对你毫无需求，这些都不是"交情"。
📉
需求消失 = 关系消失
曾经的亲密无间、不可或离，却渐行渐远。原因往往非常简单——没有需求了。

穿越百年的产业故事

丝绸之路 → SHEIN

📱
中国手机 30 年崛起
从山寨机到全球前五，中国手机产业的崛起本质是精准捕捉了不同阶段用户的核心需求，逐层升级。

脑白金抓住了"送礼焦虑"，瑞幸抓住了"咖啡平权"。看似不同的品类，背后是同一种方法论：找到真需求，构建共识。

07 / 08 交叉洞察

两本书的共振

  • 01

    需求是一切的起点

    王宁发现了潮玩背后的情感需求（满足感 + 存在感），梁宁说"因为被需求，所以有价值"。两者殊途同归——找到真需求，才有独特价值。

  • 02

    独特是最好的共识策略

    梁宁说共识是"对分歧的超越"。泡泡玛特正是用"独特"建立了新共识——潮玩不是玩具，是艺术化消费的代表。

  • 03

    模式 = 长期主义 × 实干

    梁宁的"模式"强调可持续竞争力，王宁的"尊重时间、尊重经营"是模式的最佳注脚。不靠风口，靠深度。

  • 04

    人性洞察 > 数据分析

    两本书都强调：好的商业决策来自对人性的深刻理解，而不仅仅是数据报表。乔布斯不调研市场，王宁从社群中发现 SONY ANGEL。

核心行动框架

① 找到真需求 — 不是用户说要什么，而是他们无法离开什么

② 做到独特 — 在小品类中深耕，每个小领域都能诞生伟大公司

③ 建立共识 — 让市场认可你的价值定义，而不是追随别人的

④ 形成模式 — 尊重时间，尊重经营，把独特做成可复制的系统
