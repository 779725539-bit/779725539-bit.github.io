# 程翔的个人知识库

这是我的个人线上知识库，也是一个持续更新的 AI 产品与个人认知图谱。

我会把自己在 AI 产品、Agent 时代、创业判断、硬件项目、读书笔记、产品思考和日常观察中的内容整理到这里。它不是一个单纯的资料仓库，而是一个用来沉淀判断、复盘项目、整理问题、连接想法的长期知识系统。

这个知识库主要关注几个方向：

- AI 产品如何从“工具”走向“关系、记忆、陪伴和物理在场”
- Agent 时代里，个人知识、项目经验和上下文如何被持续整理和复用
- 创业和产品推进中，如何判断一个想法是否能落地、能商业闭环、能长期维护
- AI 硬件、可穿戴设备、儿童 4G 对讲机、录音硬件、陪伴型产品等项目中的真实问题、阶段性判断和复盘
- 读书、观察、项目记录和对话内容，如何沉淀成可复盘、可继续生长的知识
- 个人长期思考如何从碎片变成结构，从记录变成判断

我希望这个网站不仅保存资料，也能呈现一个人如何在真实项目里持续形成判断：哪些想法被验证，哪些项目被推进，哪些问题暂时没做通，哪些经验可以继续复用。

项目以静态网页形式公开，内容会被整理成可直接阅读的知识条目、专题页面和项目笔记，适合快速浏览，也适合长期回看。
